# STC Solutions AI Edge

منصة الحافة الكاملة: واجهة ويب، كاميرات، موديولات حريق/دخان/مكافحة سرقة، تجربة 14 يوم بدون ترخيص.

## المكونات

| المكون | الوصف |
|--------|--------|
| **server/** | خادم الحافة (FastAPI): واجهة ويب + API + SQLite + تجربة 14 يوم |
| **ui/** | واجهة ويب: Dashboard، Cameras (إضافة + اختيار موديول)، Events، Settings، License |
| **inference/** | موديولات AI: fire, smoke, anti_theft, person (detectors.py) |
| **event_engine/** | محرك الأحداث: تحقق متعدد الإطارات، خطر، إزالة تكرار، تحريك السيرن |
| **sync/** | عميل مزامنة (Rust): config، license، events، telemetry مع السحابة |
| **ingestion/** | استقبال فيديو (Rust): RTSP، قائمة إطارات |

## تشغيل خادم الحافة (واجهة + API + كاشف تجريبي)

**بدون ترخيص لأول 14 يوم.** يعمل لوحده بدون اتصال سحابة.

```bash
cd edge/server
pip install -r requirements.txt
python run.py
```

ثم افتح المتصفح: **http://localhost:8000**

- **Dashboard**: إحصائيات، أحداث حديثة، Arm/Disarm، زر تحديث 🔄.
- **Cameras**: إضافة/تعديل/حذف كاميرا، موديولات (حريق، دخان، منع السرقة، أشخاص)، حساسية، حالة الاتصال. عند أول تشغيل تُضاف كاميرتان تجريبيتان تلقائياً.
- **البث المباشر**: لقطات حية من الكاميرات (تحديث كل 3 ثوانٍ)، يتطلب OpenCV واتصال RTSP.
- **Events**: سجل الأحداث مع فلترة (حريق، دخان، تسلل، تجمهر، أشخاص، حالة الكاميرا).
- **Settings**: إعدادات الإشعارات، واتساب، ربط الموبايل (كود الربط)، موديولات الذكاء وجدولة السرقة، إعدادات متقدمة، مخرجات وصفّارة (ريليه أو سماعات الكمبيوتر، 8 أصوات لكل نوع، مدة الصفّارة)، الموقع.
- **License**: حالة التجربة (14 يوم)، وإدخال **مفتاح الجهاز** و(اختياري) عنوان السحابة.

## تجربة 14 يوم

- عند أول تشغيل يُنشأ ترخيص تجريبي تلقائياً (14 يوم من تاريخ البدء).
- لا يلزم اتصال إنترنت أو تفعيل ترخيص خلال هذه الفترة.
- الموديولات: Fire، Smoke، Anti-theft تعمل كلها خلال التجربة.

## موديولات الكشف

| الموديول | الوصف |
|----------|--------|
| **Fire** | كشف لهب/شرر (مع نموذج ONNX في الإنتاج) |
| **Smoke** | كشف دخان |
| **Anti-theft** | دخول غير مصرح، تواجد مشبوه (intrusion, loitering). يدعم جدولة (أوقات تشغيل). |
| **Person** | اكتشاف أشخاص، عدّ، تقدير عمر ونوع (للتحليلات). |

في الكاميرا يمكن تفعيل واحد أو أكثر من: `fire`, `smoke`, `anti_theft`.

## قاعدة البيانات المحلية

تُنشأ تلقائياً في `edge/data/riskintel.db` (أو عبر متغير البيئة `EDGE_DB_PATH`).

## المزامنة مع السحابة (اختياري)

**من الواجهة:** License → "Activate with key from Cloud" → أدخل مفتاح الجهاز و(اختياري) عنوان الـ API → Save & link. الخلفية ستستخدمها تلقائياً لمزامنة الأحداث.

**أو عبر متغيرات البيئة:**

```bash
export SYNC_BASE_URL=https://api.your-domain.com
export SYNC_DEVICE_KEY=your-device-key
cd edge/sync && cargo run --release
```

خادم الحافة (Python) يقرأ أيضاً من الإعدادات المحفوظة (device_key و cloud_url) إن لم تكن المتغيرات مضبوطة.
